# Startaideia

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) versão 9.0.1.

## Passo 1

Cerifique-se que a api (back) esteja rodando na porta 3000: (http://localhost:3000/swagger-ui.html)

## Passo 2

Descompacte o arquivo dist.zip (build do front) e Copie a pasta 'startaideia' para o servidor de email (ex:nginx)
* altere as permissões para que o servidor possa ter acesso leitura/escrita em toda a pasta  

## Passo 3

Configurar o servidor apontando para o arquivo index.html

E voila...
